## üìå Project Overview
This project presents an in-depth **Exploratory Data Analysis (EDA)** of the Indian Premier League (IPL). I have analyzed match results, the impact of toss decisions, and individual player performances (both Batting & Bowling) using comprehensive datasets spanning from 2008 to 2024.

## üìä Key Insights & Visualizations

### 1. Most Successful Players (Man of the Match)
This analysis identifies the players who have dominated IPL history by winning the maximum number of 'Player of the Match' awards.
![Man of the Match Graph](image1.png) 

### 2. Toss Factor Analysis
This section explores whether winning the toss significantly correlates with winning the match, analyzing historical trends and team strategies.
![Toss Analysis Graph](image2.png)

### 3. Top Run Scorers (Orange Cap)
A detailed breakdown of the highest run-getters in the tournament, calculated using player-specific scoring data.
![Top Scorers Graph](image3.png)

### 4. Correlation Heatmap
A statistical visualization showing how different numerical variables (such as runs, wickets, and seasons) relate to one another.
![Heatmap](image4.png)
